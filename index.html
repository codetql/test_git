GIT ~

Lệnh git

git init (dùng khởi tạo git trên máy local )

git clone [nguồn git] (dùng tạo một bản sao mã nguồn lấy từ git repository)

git add hoặc git add . (dùng thêm phần sữa đổi từ máy local vào stage)

git commit -m [ghi chú cho commit] (dùng thêm phần sửa đổi từ stage vào head)

git remote push (dùng đẩy mã từ head lên git repository
